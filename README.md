# AMA-Technical-Paper

1. What are semantic elements in HTML?
```
Semantic elements clearly describe their meaning in a webpage.
 Examples: <header>, <nav>, <section>, <article>, <footer>.
```
2. What are selectors in CSS?
 ```
Selectors are patterns used to target HTML elements for styling.
Examples: element selector, class selector, ID selector, attribute selector, pseudo-classes.
```
3. What is HTML?

HTML (HyperText Markup Language) is the standard language used to structure content on the web.

4. What is the difference between italic and emphasis?

       <i> only makes text italic (visual).
        <em> adds semantic meaning and stresses importance; screen readers emphasize it.
5. What is multi-level inheritance?

When a class inherits from another derived class.
          Example: A → B → C (C inherits B, B inherits A).


6. Why do we use anchor tag in HTML?

        The <a> tag is used to create hyperlinks to webpages, sections, email, phone, or files.
7. What is a "fav" icon?

A favicon is the small icon shown in the browser tab and bookmark bar.
Added using:
```
            <link rel="icon" href="favicon.ico">
```
9. Different ways to include CSS in a webpage.

Inline CSS
```
<p style="color:red">

```
Internal CSS
```
<style> p { color:red; } </style>
```
External CSS
```
<link rel="stylesheet" href="style.css">
```

11. What is GIL in Python?

GIL (Global Interpreter Lock) allows only one thread to execute Python bytecode at a time, limiting true multithreading.


12. Difference between shallow and deep copy.

Shallow copy: Copies only references of nested objects.

Deep copy: Creates a complete copy of all nested objects.
