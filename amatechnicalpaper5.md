# AMA Technical Paper---

## 1. What is the use of `console.log` in JavaScript?
`console.log()` is used to print output or debug information in the browser’s console during JavaScript execution.

---

## 2. What is a template in Django?
A Django template is an HTML file that uses Django Template Language (DTL) to display dynamic data sent from views.

---

## 3. What are models in Django?
Models define the structure of database tables using Python classes and handle database operations through Django ORM.

---

## 4. What is WSGI in Django?
WSGI (Web Server Gateway Interface) is a standard that allows Django to communicate with web servers like Gunicorn or Apache.

---

## 5. What are static files in Django?
Static files are files like CSS, JavaScript, and images used to style and add interactivity to Django web applications.

---

## 6. What is the difference between `render()` and `redirect()`?
- `render()` → Returns an HTML template with context data  
- `redirect()` → Sends the user to a different URL

---

## 7. What is `manage.py` used for?
`manage.py` is a command-line utility used to run the server, create apps, apply migrations, and manage Django projects.

---

## 8. What does `migrate` do in Django?
`migrate` applies database changes based on migration files and syncs models with the database schema.

---

## 9. What is `transaction.atomic()`?
`transaction.atomic()` ensures that a group of database operations either all succeed or all fail, maintaining data consistency.

---

## 10. What is the difference between a Django project and a Django app?
- Project → The entire Django setup (settings, URLs, apps)  
- App → A specific module that performs a single function (e.g., accounts, posts)

---

## 11. What is a model field in Django?
A model field defines the type of data stored in a database column, such as `CharField`, `IntegerField`, or `DateTimeField`.
