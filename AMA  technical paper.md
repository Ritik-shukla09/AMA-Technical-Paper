1. How to rename a table in SQL?
* Use the RENAME or ALTER TABLE command to change a table’s name.
```
Example:

ALTER TABLE old_name RENAME TO new_name;
```
2. What is a module in Python?

* A module is a Python file containing functions, classes, and variables that can be imported and reused in other programs.

 * int, float, complex

4. What are SQL wildcards?

Wildcards are special characters used with LIKE to match patterns.

% → matches any number of characters

_ → matches one character

5. Difference between generators and iterators?

Generator: Creates iterators using yield, memory-efficient, creates values on the fly.

Iterator: Any object with __iter__() and __next__() methods.

8. Multithreading vs Multiprocessing in Python

Multithreading: Multiple threads, shared memory, good for I/O tasks but limited by GIL.

Multiprocessing: Multiple processes, separate memory, best for CPU-heavy tasks.

9. SSR vs CSR

SSR (Server-Side Rendering) → HTML generated on server; faster first load.

CSR (Client-Side Rendering) → Browser renders using JS; slower first load but faster navigation.

10. What is duck typing?

Python concept: If an object behaves like a type, it is treated as that type, regardless of its actual class.

11. Difference between UNION and UNION ALL

UNION → combines results and removes duplicates.

UNION ALL → combines results without removing duplicates.

12. Which tag is used for internal CSS?

Internal CSS is written inside the ```<style> tag in the <head> section.```
